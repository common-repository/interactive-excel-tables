== Description ==

This plugin allows you to import excel file into wordpress.

Documentation
To add table, go to page "Excel tables" in admin panel.

== Installation ==

Installing "Excel tables" can be done either by searching for "Excel tables" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
